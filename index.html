How to use

Get list of themes

GET: {LINK}request/themes
Warning: Themes with -h suffix is a NSFW!

Demo image

GET: {LINK}request/demo?theme=&smoothing=

Any number image

GET: {LINK}request/number?value=&theme=&lead_zeros=&max_length=&smoothing=

Register your counter

POST: {LINK}request/take_key?key=&password=&salt=
Place for request answer

Delete your counter

DELETE: {LINK}request/revoke_key?key=&password=&salt=
Place for request answer

Counter image

GET: {LINK}request/image@?theme=&lead_zeros=&max_length=&smoothing=&do_inc=

Counter record

GET: {LINK}request/image@&do_inc=
Place for request answer

Change counter value

DELETE: {LINK}request/set@key?password=&salt=&value=
Place for request answer
